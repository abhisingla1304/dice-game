# 🎲 Dice Game (Made in Python):- 

This is a simple dice rolling game built using Python.

##  Key Features:- 

1. Generates random dice numbers.
2. User can roll multiple times.
3. Simple and interactive terminal output.
4. you can select number of players according to your need.


##  How to Run:- 

1. Open terminal
2. Run:

  command- python main.py
